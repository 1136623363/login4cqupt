# login4cqupt
Login the camplus network for CQUPT.
 
### Usage
Support Windows & Linux & Mac.
```shell
git clone https://github.com/ourongxing/login4cqupt.git
cd login4cqupt
# then
python main.py 1658xxx passxxxx -o cmcc
# help
python main.py -h
```
